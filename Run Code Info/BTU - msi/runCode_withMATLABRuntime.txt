1) When compiling:

i) HOME --> Parallel --> Create and Manage Clusters (Cluster Profile Manager) --> Export --> save "local.settings"

ii) In addition to the main file "buildSaveVideoFrameMasks_withThreshold_and_Fingerprint", add also the folders "commonFunction" into "Files required ...", and "Idecod", "jm_16.1_xmltrace_v1.5" and "local.settings" into "Files installed ..."

2) After compilation is finished, go inside the "for_redistribution_files_only" folder

3) Make sure that the file "local.settings" and the folders "Idecod" and "jm_16.1_xmltrace_v1.5" are inside the folder "for_redistribution_files_only".

4) Create the folders "output" and "temp" inside the "for_redistribution_files_only" folder

5) run code option 1: inside the folder "for_redistribution_files_only" run the following command
