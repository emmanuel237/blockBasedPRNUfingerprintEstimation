Step 1: In terminal, run the following command:
cat /etc/exports

Step 2: In the opening file, write the following info:
#exporting the disk Data2 and VISION folder in Data1 disk
/media/emmanuel/Data2 *(rw, sync,no_root_squash,no_subtree_check)
/media/emmanuel/Data1/VISION *(rw, sync,no_root_squash,no_subtree_check)

Step 3: After closing the file, run the following commands in the terminal:
exportfs -a
If required, restart the nfs server as follows:
sudo service nfs-kernel-server restart
